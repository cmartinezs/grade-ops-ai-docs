# User Stories

- As a programming instructor, I want a rubric draft from an assessment brief so I can start faster.
- As a teacher, I want grading suggestions tied to the rubric so I can review rather than score from scratch.
- As a tutor, I want feedback drafts for each learner so I can personalize them quickly.
- As an academy operator, I want learning gap summaries across a cohort so I can intervene earlier.
- As an educator, I want recovery activity suggestions so I can help students close specific gaps.
- As a teacher, I want a report for each assignment or cohort so I can communicate outcomes clearly.
