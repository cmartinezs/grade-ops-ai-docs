# Workflows

## Core workflow
1. Teacher defines assessment intent and constraints
2. Assessment and rubric agents prepare first drafts
3. Teacher reviews and approves the rubric
4. Submissions are received and organized
5. Grading agent produces structured scoring suggestions
6. Teacher reviews and finalizes judgments
7. Feedback, learning-gap, and recovery agents prepare learner outputs
8. Teacher report agent summarizes cohort outcomes
9. Ops agent stores logs and evidence artifacts

## Control principle
Every high-impact output should have an explicit teacher review step.
