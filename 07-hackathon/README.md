## Required Evidence Summary

The submission must disclose:

- Revenue earned and revenue by month.
- Related-party vs. arms-length revenue.
- Total operating costs.
- Marketing and customer acquisition spend (even if zero).
- User count and evidence.
- Product evidence (screenshots, demo, logs).
- Agent / API logs from a live product deployment.

See [`evidence-checklist.md`](evidence-checklist.md) for the full collection plan.
