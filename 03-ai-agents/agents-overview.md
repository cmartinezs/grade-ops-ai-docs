# Agents Overview

GradeOps AI uses specialized agents instead of one generic assistant.

## Design principles
- One clear responsibility per agent
- Structured handoffs between agents
- Teacher approval at key checkpoints
- Logs for every meaningful operation

## Agent set
- Assessment agent
- Rubric agent
- Grading agent
- Feedback agent
- Learning gap agent
- Recovery agent
- Teacher report agent
- Ops agent
