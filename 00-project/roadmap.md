# Roadmap

## Phase 1: Hackathon MVP
- Define core workflow for one programming assessment type
- Support rubric drafting, grading assistance, feedback drafts, and teacher reports
- Capture agent logs, usage events, customer interviews, and payment evidence

## Phase 2: Pilot operations
- Run with a small set of real educators
- Improve review controls and auditability
- Add recurring cohort and assignment management

## Phase 3: Expansion
- Support multiple assessment formats
- Add analytics across cohorts and instructors
- Expand beyond the initial programming education wedge
